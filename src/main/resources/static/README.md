# 🇯🇵 일본 여행 LMS 웹사이트

일본 여행을 준비하는 학습자들을 위한 종합 학습 관리 시스템(Learning Management System)입니다. 일본어 학습, 문화 이해, 여행 정보 등을 체계적으로 학습할 수 있으며, Unity WebGL 기반의 가상 체험 기능도 제공합니다.

## 🎯 프로젝트 개요

**프로젝트명**: 일本 Travel LMS  
**목표**: 일본 여행을 준비하는 사람들을 위한 온라인 학습 플랫폼  
**메인 컬러**: 흰색(#FFFFFF)과 빨간색(#C1272D) - 일본 국기를 상징

### 핵심 가치
- 📚 체계적인 일본 여행 준비 학습
- 🎮 Unity WebGL 기반 가상 체험
- 📊 학습 진도 관리 및 통계
- ✅ 출석 체크를 통한 학습 습관 형성

## 🌟 주요 기능

### 1. 로그인 시스템 (index.html)
- 중앙 배치된 깔끔한 로그인 UI
- 일본 테마의 그라데이션 배경 (빨간색)
- 아이디/비밀번호 입력 및 로그인 유지 기능
- 회원가입 링크 제공

### 2. 선택 화면 (select.html)
- 로그인 후 표시되는 분할 화면
- **왼쪽 패널**: 학습하기 (LMS 메인으로 이동)
- **오른쪽 패널**: 체험하기 (Unity WebGL로 이동)
- 호버 효과와 부드러운 애니메이션

### 3. 메인 LMS 페이지 (main.html)
- 헤더: 로고, 네비게이션, 사용자 정보, 로그아웃
- 히어로 섹션: "Discover Japan" 메시지와 행동 촉구 버튼
- Featured Destinations: 도쿄, 교토, 오사카, 나라, 고베 소개
- 학습 통계: 수강 강좌, 진도율, 출석일, 학습 시간

### 4. 온라인학습 (online-learning.html)
- 강좌 카테고리 필터링 (전체/일본어/문화/여행)
- 6개의 다양한 강좌 제공:
  - 일본어 기초 회화
  - 일본어 중급
  - 일본 문화 이해
  - 도쿄 여행 가이드
  - 일본 교통수단 이용법
  - 일본 음식 문화
- 각 강좌별 진도율 표시
- 학습 시작 버튼

### 5. 출석체크 (attendance.html)
- 일일 출석 체크 기능
- 출석 통계: 총 출석일, 연속 출석일, 이번 달 출석, 출석률
- 이번 달 출석 캘린더 시각화
- 최근 출석 기록 테이블
- 로컬 스토리지를 활용한 데이터 저장

### 6. 나의 강의실 (my-classroom.html)
- 학습 통계 대시보드
- 수강 중인 강좌 목록 및 진도율
- 완료한 강좌 목록
- 최근 학습 활동 타임라인
- 강좌 이어하기/복습하기 기능

### 7. 이용안내 (guide.html)
- 시작하기 가이드
- 주요 기능 소개
- 학습 팁 제공
- 자주 묻는 질문 (FAQ)
- 문의하기 기능

## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업

  - 애니메이션 및 트랜지션
  - 반응형 디자인 (@media queries)

  - DOM 조작
  - 이벤트 처리
  - LocalStorage API
  - SessionStorage API
  - Date 객체 활용

### 디자인
- **컬러 스킴**:
  - Primary Red: #C1272D
  - Secondary Red: #E74C3C
  - White: #FFFFFF

  - Dark Gray: #333333
- **폰트**: Segoe UI, Malgun Gothic, Apple SD Gothic Neo
- **이미지**: Unsplash API

## 🚀 시작하기

### 1. 로그인
- 메인 페이지(index.html)에서 아이디와 비밀번호 입력
- 테스트용: 아무 값이나 입력 가능 (인증 없음)

### 2. 메뉴 선택
- 로그인 후 선택 화면 표시
- **학습하기**: LMS 메인 페이지로 이동
- **체험하기**: Unity WebGL 체험 (별도 URL 설정 필요)

### 3. 학습 시작
- 상단 네비게이션에서 원하는 메뉴 선택
- 온라인학습: 강좌 선택 및 학습 시작
- 출석체크: 매일 출석 버튼 클릭
- 나의 강의실: 학습 진도 확인

### 4. 주요 단축키
- **ESC**: 이전 페이지로 이동 (선택 화면, 메인 페이지)

## 💾 데이터 저장

### SessionStorage
- `isLoggedIn`: 로그인 상태 (true/false)
- `username`: 사용자 이름

### LocalStorage
- `username`: 로그인 유지 시 아이디 저장
- `attendanceData`: 출석 데이터 (JSON 형태)
  - totalDays: 총 출석일
  - consecutiveDays: 연속 출석일
  - thisMonth: 이번 달 출석
  - lastCheckIn: 마지막 출석일
  - checkedDates: 출석한 날짜 배열
